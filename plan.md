# 英文單字學習工具 開發計畫 (plan.md) - 優化版

## 1. 專案目標
開發一個現代化 Windows 視窗介面的英文單字學習工具，內建 3000 個高頻單字，並具備發音快取、學習追蹤與自動備份功能。

## 2. 技術與架構優化
- **GUI 框架**：CustomTkinter (現代化風格)
- **資料庫**：SQLite (含自動備份機制)
- **發音系統**：gTTS + 語音快取 (儲存於 `assets/audio/`)，支援美/英音切換。
- **架構模式**：採用類 MVC 設計，將 UI 元件與資料邏輯拆分，並引入 **AudioManager** 統一管理音訊流。
- **異步處理**：網路請求、語音生成以及 **資料庫搜尋載入** 採用多執行緒 (Threading) 避免 UI 凍結。
- **效能機制**：實作 **防抖動 (Debouncing)** 減少無效查詢，並在資料庫建立單字索引提升速度。
- **路徑與日誌**：使用 `pathlib` 進行跨平台路徑管理，並整合 `logging` 系統追蹤運行異常。

## 3. 檔案結構 (Refactored)
- `main.py`: 程式啟動入口。
- `app.py`: 主視窗控制與導航邏輯。
- `db_manager.py`: 資料庫操作與備份邏輯。
- `views/`: 存放各個頁面元件 (主頁、清單、測驗、設定)。
- `utils/`: 存放語音管理 (`audio_manager.py`)、路徑工具與 TTS 下載器。
- `assets/audio/`: 存放單字 MP3 快取。
- `tests/`: 存放資料庫與邏輯單元測試。
