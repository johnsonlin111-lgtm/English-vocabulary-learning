# 英文單字學習工具 - 需求規格說明書 (spec.md)

## 1. 專案簡介
這是一個專為學習者設計的英文單字學習工具，結合了考試必備的「核心詞庫」與「個人化學習」功能。透過多樣化的測驗模式、閃卡練習、**真人音質發音**、**語境例句**與進度追蹤，幫助使用者更有效地掌握並記憶英文單字。

## 2. 使用者故事 (User Stories)
- **考試準備者**：身為一名正準備托福或雅思考試的學生，我希望能直接取得系統內建的高頻率單字庫，讓我不需要花時間四處蒐集必背單字。
- **自主學習者**：身為一名英文學習者，當我在閱讀文章或看影片時遇到新單字，我希望能手動將它們加入我的清單中，並能隨時修正或刪除打錯的內容。
- **聽力練習者**：身為一名重視發音的學習者，我希望能聽到每個單字的正確讀音，幫助我在記住意思的同時也能學會如何發音。
- **高效複習者**：身為一名想要鞏固記憶的學習者，我希望能標記哪些單字已經背熟、哪些還很生疏，並能透過例句理解單字在真實句子中的用法。
- **數據守護者**：身為一名長期學習者，我希望能備份我的單字庫，避免因為換電腦或程式遺失而導致辛苦建立的自訂單字消失。

## 3. 功能需求 (Functional Requirements)

### 3.1 單字庫管理 (CRUD)
- **內建核心詞庫**：預載 3000 個托福 (TOEFL) 與雅思 (IELTS) 的高頻率單字（含基本中譯、**音標**、**發音音訊**與**至少一條例句**）。
- **完全管理功能**：
    - **新增**：手動輸入單字、中文翻譯、音標及例句。
    - **編輯**：修正已存在單字的內容（包含內建單字也可自訂筆記與例句）。
    - **刪除**：移除不再需要的單字。
- **檢視與搜尋**：列出所有單字，並支援關鍵字搜尋。

### 3.2 語音與發音支援
- **文字轉語音 (TTS)**：支援單字與例句的自動發音功能，使用高品質合成語音或播放內建音訊。
- **發音按鈕**：在列表、閃卡與測驗介面中提供「發音圖示」，點擊後播放讀音。
- **離線支援**：若無網路且無快取，系統應提供友善提示或嘗試呼叫系統內建 TTS。

### 3.3 學習進度與標記
- **掌握度標籤**：每個單字可標記為「學習中 (Learning)」或「已掌握 (Mastered)」。
- **鍵盤快速鍵與 UI 操作**：
    - **閃卡模式**：支援 `Space` 翻面、`1` 記住、`2` 不熟。介面提供對應的實體按鈕與「結束練習」按鈕。
    - **測驗模式**：支援 `1-4` 數字鍵作答。提供「結束並查看報告」功能。
- **搜尋優化**：支援大小寫不敏感、自動修剪空格，並具備非同步搜尋與防抖動機制以確保流暢度。

### 3.4 學習與練習模式
- **單字閃卡 (Flashcards)**：
    - 正面顯示英文單字與音標，並可選自動播放發音。
    - 翻面後顯示中文意思、**例句**及掌握度狀態。
- **測驗與回饋**：
    - **結算報告**：測驗結束後顯示正確率百分比。
    - **錯題回顧**：結算畫面列出本次答錯的單字清單，可直接點擊發音複習。
- **使用者設定**：
    - 支援儲存介面主題 (深色/淺色)、預設發音人 (美/英) 與語速。

- **多樣化測驗模式**：
    - **翻譯測驗**：雙向英翻中、中翻英。
    - **拼字測驗 (Spelling)**：顯示中文、**播放單字發音**並提供部分提示，填寫完整拼法。
    - **填空測驗 (Cloze)**：根據句中語境（顯示例句）填入正確單字。
- **強化練習 (Mistake Collection)**：系統自動記錄測驗中答錯的單字，並提供獨立的「錯題練習」區。

### 3.5 資料安全與備份
- **資料持久化**：所有變更即時儲存在本地資料庫或檔案中。
- **匯出/匯入**：支援匯出單字庫（含自訂例句）為 JSON 或 CSV 格式。

## 4. 成功標準 (Success Criteria)
- **功能完備性**：實現發音、例句展示、單字管理等核心功能。
- **音訊品質**：發音功能清晰無延遲，支援多數常用單字的正確讀音。
- **語境關聯性**：內建單字的例句需符合其常用語意，幫助使用者理解用法。
- **數據安全性**：使用者自訂的例句與發音標記在程式重啟後完整保留。
- **效能表現**：在大數據量下，載入發音檔案或生成 TTS 時介面不應凍結。
